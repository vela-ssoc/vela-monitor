# 本地告警模块 TODO

## 功能概述
- 监控本地系统简单指标
- 支持LUA层一键配置告警规则
- 触发告警通知,SSOC或者SIEM等

## 使用场景
适用于轻量级的本地监控需求，包括：
- CPU使用率告警
- 内存使用告警
- 磁盘空间告警

## 与专业监控系统对比
| 特性        | 本地告警 | Prometheus |
|------------|---------|------------|
| 部署复杂度   | 简单     | 复杂       |
| 功能完整性   | 基础功能 | 完整方案   |
| 适用规模     | 单机    | 分布式集群  |

**注**：复杂监控场景建议使用专业监控系统如Prometheus
